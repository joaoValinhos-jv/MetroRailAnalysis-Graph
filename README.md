# MetroRailAnalysis-Graph
This code is made due to the educational purposes for studying how the graphs theory can help to map the São Paulo subways and trains.
It is based on data analyses, there are two .csv files, one containing all the statios analysed and the other containg how the stations connect to each other.
The graphs can be seeing in Gephi Lite (https://lite.gephi.org/v1.0.2/).
The script for giving the eight shape to the graph is right below, and it must be pasted in the custom layout tab on the site.
